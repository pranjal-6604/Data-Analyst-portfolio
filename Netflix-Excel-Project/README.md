# Netflix Data Analysis Dashboard

# Objective 

Analyze Netflix dataset to identify trends in content distribution, genres, ratings, and viewing patterns to derive meaningful business insights. 

## Steps Performed 

- Performed data cleaning by handling missing values and removing duplicates
- Transformed raw data using Excel functions for better analysis
- Conducted Exploratory Data Analysis(EDA) using Pivot Tables
- Created KPIs and built an interactive dashboard using charts and filters

## Key Insights 

- Majority of content is Movies compared to TV shows
- Certain genres dominate the platform's content library
- Content addition increased significantly in recent years
- Ratings distribution shows concentration in specific categories

## Tools Used 

Microsoft Excel(Pivot Tables, Charts, Filters, Function)

## Output 

Developed an interactive Excel dashboard to visualize content trends and support data-driven insights.
