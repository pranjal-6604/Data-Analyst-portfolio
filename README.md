## Data Analytics Portfolio

This repository contains my Data Analytics projects built using Excel and Power BI.

It demonstrate how raw data is cleaned, analyzed, and transformed into meaningful business insights using dashboards and visaul reports.

## Projects

### Ecommerce Sales Analysis (SQL + Power BI)
- Designed a relational database with multiple tables (customers,orders,products,payments).
- Performed data cleaning and transformation in SQL to ensure consistency and integrity.
- Wrote advanced SQL queries using joins, aggregations, subqueries, and window function.
- Analyzed revenue, customer behavior, and product performance.
- Built an interactive Power BI dashboard with KPIs, trends, and category-level insights.

### Netflix Data Analysis (Excel)
- Performed content trend analysis to identify in genres, release years, and content distributions.
- Developed Key performance indicators(KPIs).
- Built an interactive dashboard in Excel using pivot tables, charts, and slicers for dynamic filtering.

### Retail Sales Analysis (Power BI)
- Conducted sales performance analysis to evaluate revenue trends and product/category contribution
- Analyzed customer behavior including purchasing patterns and regional variations.
- Designed a Power BI dashboard with KPIs (Revenue, Profit, AOV).

## Skills Demonstrated
- SQL(Joins, Aggregations, Window Functions) - Used for structured data analysis and extracting insights.
- Data Cleaning - Handling missing values,correcting inconsistencies, and preparing structured datasets.
- Exploratory Data Analysis (EDA) - Identifying patterns, trends, and anomalies in data.
- Data Visualization - Creating dashboards and charts for clear communication of insights.
- Business Intelligence - Converting raw data into meaningful business insights for decision-making.
